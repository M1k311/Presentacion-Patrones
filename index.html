¿Por qué patrones estructurales?

El Reto de Escalar

El problema

La herencia rígida genera jerarquías exponencialmente complejas conforme el sistema crece. Modificar una clase puede romper decenas de subclases.

La solución

Composición sobre herencia: ensamblar objetos para crear estructuras que cambian en tiempo de ejecución sin alterar el código base.

Decorator

Añade responsabilidades dinámicamente sin modificar la clase original.

Facade

Interfaz simple que oculta la complejidad de subsistemas internos.

Flyweight

Comparte estado entre miles de objetos para ahorrar memoria RAM.

Proxy

Controla el acceso al objeto real: seguridad, caché y lazy loading.

Patrón 01 / 04

Decorator (Wrapper)

Concepto técnico

Asigna nuevas responsabilidades a objetos en tiempo de ejecución. Envuelve el objeto original preservando su interfaz sin subclasificar.

Analogía: Ponerse ropa. Un suéter añade calor sin cambiar quién eres. Puedes quitarlo o añadir una chaqueta encima.
Java I/O StreamsUber (capas)HTTP Middleware

# Componente base
class CafeBase:
    def costo(self):
        return 20.0

# Decorador concreto
class DecoradorLeche:
    def __init__(self, cafe):
        self.cafe = cafe   # Envuelve
    def costo(self):
        return self.cafe.costo() + 5.0

class DecoradorVainilla:
    def __init__(self, cafe):
        self.cafe = cafe
    def costo(self):
        return self.cafe.costo() + 8.0

# Composición dinámica
mi = CafeBase()
mi = DecoradorLeche(mi)
mi = DecoradorVainilla(mi)
# mi.costo() → 20 + 5 + 8 = 33

Facade (Fachada)

Concepto técnico

Proporciona una interfaz simplificada a un subsistema complejo. Desacopla al cliente de los detalles internos de implementación.

Analogía: Un operador de Call Center: hablas solo con una persona, que coordina en silencio con logística, pagos y almacén.

class FachadaTienda:
    def __init__(self):
        self.inv = Inventario()
        self.banco = Banco()
        self.envio = Envio()

    # El cliente solo llama ESTO:
    def comprar(self, prod, user):
        if not self.inv.hay_stock(prod):
            raise Exception("Sin stock")
        self.banco.cobrar(
            user.tarjeta, prod.precio
        )
        self.envio.programar(
            user.direccion, prod
        )
        return "¡Compra exitosa!"

# Uso → solo 1 línea visible:

Patrón 03 / 04

Flyweight (Peso Ligero)

Concepto técnico

Reduce memoria compartiendo el estado intrínseco (inmutable) entre miles de objetos similares, separándolo del estado extrínseco (único por objeto).

Analogía: Sellos de oficina. No creas un molde nuevo por cada hoja. Usas un solo sello y cambias la posición del papel.
Google MapsLectores PDFGame engines

# Estado intrínseco (compartido)
class TipoArbol:
    def __init__(self, nombre, tex):
        self.nombre = nombre
        self.textura = tex  # ¡Pesado!

# Fábrica Flyweight (caché)
class FabricaArboles:
    _tipos = {}
    def obtener(self, nom, tex):
        key = f"{nom}_{tex}"
        if key not in self._tipos:
            self._tipos[key] = TipoArbol(nom, tex)
        return self._tipos[key]

# Estado extrínseco (único por árbol)
class Arbol:
    def __init__(self, x, y, tipo):
        self.x, self.y = x, y
        self.tipo = tipo  # solo referencia

# 10,000 árboles → 1 TipoArbol
Patrón 04 / 04

Proxy (Intermediario)

Concepto técnico

Sustituto que controla el acceso al objeto real. Añade lógica antes/después sin modificar el objeto: seguridad, caché, lazy loading.

Analogía: Un Asistente Personal: filtra llamadas y solo conecta al jefe si tienes permiso. El jefe nunca sabe quién intentó llamar.
Netflix CDNAWS CloudFrontNginxORM Lazy
Demo — Control de Acceso
// Selecciona un rol e intenta acceder...
class ServidorReal:
    def acceder(self):
        return "Datos sensibles"

class ProxySeguridad:
    _real = None  # Lazy loading
    _perms = {
        "Admin": ["leer", "escribir"],
        "Usuario": ["leer"],
        "Visitante": [],
    }
    def acceder(self, rol):
        ps = self._perms.get(rol, [])
        if "leer" in ps:
            if not self._real:
                self._real = ServidorReal()
            return self._real.acceder()
        return "Acceso denegado"
