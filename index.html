Composición y Estructura

El Desafío

Al escalar sistemas, la herencia rígida genera jerarquías inmanejables. Los patrones estructurales proponen ensamblar objetos para mantener la flexibilidad.

La Estrategia

Se utiliza la composición sobre la herencia para crear estructuras eficientes que puedan cambiar en tiempo de ejecución.

1. Decorator (Wrapper)

Concepto Técnico

Asigna nuevas responsabilidades a objetos de forma dinámica. Evita crear una subclase para cada combinación de funciones.

Analogía: Como ponerse ropa. Un suéter añade calor sin cambiar quién eres; puedes quitarlo o añadir una chaqueta encima.
Uber (Capas de red) Java I/O Streams 
class DecoradorLeche:
    def __init__(self, cafe):
        self.cafe = cafe
    def costo(self):
        return self.cafe.costo() + 5.0

2. Facade (Fachada)

Concepto Técnico

Proporciona una interfaz simplificada a un conjunto complejo de clases. Desacopla al cliente de la implementación interna.

Analogía: Como un operador de Call Center. Tú solo hablas con una persona, y ella se encarga de coordinar con logística, pagos y almacén.
Stripe (Pagos) Mercado Pago 
class FachadaTienda:
    def comprar(self):
        if self.inv.hay_stock():
            self.banco.pago()
            self.envio.mandar()

3. Flyweight (Peso Ligero)

Concepto Técnico

Reduce el consumo de memoria compartiendo el "estado intrínseco" inmutable entre miles de objetos similares.

Analogía: Los sellos de oficina. No creas un molde nuevo para cada hoja; usas un solo sello y solo cambias la posición del papel.
Google Maps Lectores PDF 
class Fabrica:
    @classmethod
    def obtener_sello(cls):
        if not cls._sello: 
            cls._sello = Sello()
        return cls._sello

4. Proxy (Intermediario)

Concepto Técnico

Sustituto que controla el acceso al objeto real. Permite validación de seguridad y Lazy Loading (carga diferida).

Analogía: Un Asistente Personal. Filtra las llamadas y solo te comunica con el jefe si es urgente o tienes una cita previa.
Netflix (Video Proxy) AWS CloudFront 
class ProxySeguridad:
    def peticion(self, user):
        if user == "Admin":
            self.real.acceder()
        else:
            print("Denegado")
